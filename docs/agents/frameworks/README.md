# Agent Frameworks

any-agent supports multiple agent frameworks, allowing you to use the framework that best fits your needs while maintaining a consistent interface.

## Supported Frameworks

- [Agno](agno.md)
- [Google ADK](google_adk.md)
- [Langchain](langchain.md)
- [LlamaIndex](llama_index.md)
- [OpenAI Agents SDK](openai.md)
- [smolagents](smolagents.md)
- [TinyAgent](tinyagent.md)

Each framework has its own strengths and use cases. Choose the one that aligns with your project requirements and existing dependencies.
